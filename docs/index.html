Why ModelKit

Type-safe

340+ OpenRouter models with generated feature IDs and autocomplete.

Zero downtime

Update models and params in production without redeploys.

React UI

Studio connects to your backend; non-engineers can manage overrides.

How it works

getModel(featureId, fallback)

→ In-memory cache (60s) → Redis override → or return fallback. Your app keeps running even if Redis is down.

Studio

A React UI that points at your backend URL. List overrides, set/clear them, and see live SDK examples. Hono and Express only.
